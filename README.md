# kimmane-leaderboard
Manage the leaderboard for Kimmane Monthly Medal Match.

## Getting started
1. Open `index.html` in your browser.
2. Add monthly medal results for Net Stroke Play and Stable Ford.
3. Review the leaderboards and Golfer of the Year highlight.

All data is stored in your browser (local storage). Use the Download/Upload
buttons in the app to back up or restore the leaderboard.
